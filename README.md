# A Puzzle A Day

一天一个谜，一个谜一天

## install

```
pip install OnePuzzle
```

## usage

```
$ one_puzzle -h
Usage: one_puzzle [OPTIONS] MONTH DAY

Options:
  -h, --help           Show this message and exit.
  -d, --debug BOOLEAN  show debug message.
  -a, --all BOOLEAN    get all result.
```
