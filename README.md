## install

```
pip install OnePuzzle
```

## usage

```
$ one_puzzle -h
Usage: one_puzzle [OPTIONS] MONTH DAY

Options:
  -h, --help   Show this message and exit.
  -d, --debug  show debug message.
  -a, --all    get all result.
  -n, --nox    no screen show.
```
